# 用隐私交换便捷？李彦宏说，中国人愿意！

**”中国人对隐私问题的态度更加开放，相对来说也没那么敏感。如果他们可以用隐私换取便利、安全或者效率，在很多情况下他们就愿意这么做。“**

百度公司董事长兼CEO李彦宏26日在钓鱼台举办的中国发展高层论坛2018年会上发表主题演讲引起关注。
**“中国人愿意用隐私交换便捷和效率”**  

大量的数据是中国人工智能发展的得天独厚条件，但数据是有隐私的。当李彦宏认为，百度非常重视隐私问题，以及包括数据的保护。在过去几年，中国也一直在加强相关的法律法规建设。  

“但中国人更加开放或者对隐私问题没有那么敏感，如果说他们愿意用隐私来交换便捷性或者效率的话，很多情况下他们愿意这么做的。”李彦宏说。

但李彦宏表示，百度要遵循一系列原则，如果认为用数据会让所有者受益，而且他也授权使用他，我们会使用。百度基本的做法是，数据方面有该做什么，不该做什么。
